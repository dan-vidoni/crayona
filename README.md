# Crayona

Browser-based proof of concept for a hand-drawn scene editor.

The current POC uses `RoughJS` to render hand-drawn-looking shapes in a canvas.

## Current scope

- Create a scene on a canvas
- Free draw strokes
- Draw a line
- Draw a circle
- Draw a square
- Clear the canvas

## Run locally

Because this is a static app, you can open `index.html` directly in a browser.

If you prefer a local server:

```bash
python3 -m http.server 4173
```

Then open `http://localhost:4173`.
